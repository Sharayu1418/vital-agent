# Demo video script (3 min, one continuous take, no slides)

Prep: fresh browser profile (clean session), backend + frontend running,
a 3-week sleep CSV ready, DEBUG_ENDPOINTS on for the finale.

| ~time | Action | Say |
|---|---|---|
| 0:00 | Landing page, starters visible | "VITAL is one place for sleep, energy, activities, ideas and people — agents that know you, instead of twelve tabs of search." |
| 0:15 | Upload sleep CSV | "First, my real sleep data — parsed and stored per-user." |
| 0:30 | "I slept terribly all week, what's actually going on?" | "The supervisor routes this to the sleep agent — which writes pandas code and runs it in an isolated sandbox. Watch the status line." |
| 1:00 | Insight arrives with real numbers | "Real sleep debt, computed by generated code that a static gate screened first." |
| 1:15 | "I've got energy tonight though — something active but cheap?" | "Different agent: live weather + real venues with links. It remembered my budget from earlier — no re-asking." |
| 1:45 | "Find people who do this regularly" | "People Connector: communities and events, each tied to what it knows about me." |
| 2:05 | "OK, plan my weekend around all of it" | "The planner drafts — and pauses. Nothing touches my calendar without me." |
| 2:20 | Type edit: "move the climb to Sunday morning" | "Revision, paused again…" |
| 2:30 | Approve → confirmation message | "…approve, and only now does it commit. That gate is graph topology, not a prompt." |
| 2:45 | Open "What VITAL knows" drawer, delete one fact | "Everything it learned is visible and deletable." |
| 2:55 | (optional) /debug/state showing routing path | "Every hop is traceable. That's the whole point — agents you can inspect." |

Retakes: expect 3-4. Keep the failed takes — the best blog material is in them.
